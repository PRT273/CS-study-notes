第五章 数组



5.1：一维数组



一维数组的定义：

一般形式如下：数据类型符 数组名1\[长度1],数组名2\[长度2],...;

“数据类型符”是指数组元素的数据类型。“长度”必须是整型的常量表达式，其中不能出现变量。

下标是从零开始顺序编号。例：

int a\[10];

float score\[50];

char name\[20];



5.1.2：一维数组的引用

引用方法：数组名\[下标]。

保证数组下标不越界十分重要!

一维数组的内存字节数=数组长度x每一元素占用的字节数。



5.1.3：一维数组的初始化

格式如下：数据类型符 数组名\[长度]=\[初值表],...;。

完全初始化：int a\[5] = {1, 2, 3, 4, 5};。

部分初始化（剩余元素自动为0）：int b\[5] = {1, 2}; 等价于：{1, 2, 0, 0, 0}。

自动推断数组长度：int c\[] = {3, 6, 9}; 自动判断长度为3。



5.2：二维数组



5.2.1：二维数组的定义



形式：数据类型 数组名\[行数]\[列数];

例：int a\[3]\[4];   // 3 行 4 列，共 12 个 int 元素。

内存布局：按行连续存储。



5.2.2 二维数组的引用

方法如下：数组名\[行下标]\[列下标]。

合法取值范围是（行列）0到行列-1.



5.2.3 二维数组的初始化

完全初始化：int a\[2]\[3] = {

&nbsp;   {1, 2, 3},

&nbsp;   {4, 5, 6}

};



部分初始化：int a\[2]\[3] = {1, 2, 3};

等价于：

1 2 3

0 0 0

一行写法：int a\[2]\[3] = {1,2,3,4,5,6};

5.3 字符数组和字符串



字符串是有若干个字符组成的，若干的字符串可以用若干个一维字符存放，也可以用一个二维字符存放。



5.3.1 字符数组

一维数组形式如下：char 数组名\[数组长度]={初值表};

二维数组形式如下：char 数组名\[行长度]\[列长度]={{初值表},{初值表},...,{初值表};



字符数组的定义：char str\[6];

逐个初始化：char str\[6] = {'H', 'e', 'l', 'l', 'o', '\\0'};



字符数组的输入和输出：一维字符使用scanf（）和printf（）函数，使用gets（）和puts（）函数。



5.3.2 字符串

字符串常量初始化：char str\[] = "Hello";



字符串输入和输出：printf("%s", str);
scanf("%s", str); (不安全！）
