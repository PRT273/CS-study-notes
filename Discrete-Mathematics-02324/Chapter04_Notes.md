第四章 集合



4.1 集合的概念



集合是最基本的离散结构，是由一些确定的、互不相同的对象组成的整体。

组成集合的对象称为**元素。**

集合通常用 大写字母 表示：A, B, C。

元素通常用 小写字母 表示：a, b, x。



元素与集合的关系:

a ∈ A：a 是集合 A 的元素

a ∉ A：a 不是集合 A 的元素



集合的表示方法：

列举法、描述法、图示法。



常见特殊集合：

空集：∅

不含任何元素的集合



单元素集：

只含一个元素的集合，例如 {a}



全集：

讨论问题时所限定的所有对象构成的集合，常记为 U

两个集合 A、B 相等，当且仅当：

A 的每个元素都属于 B

B 的每个元素都属于 A

记作：A = B



子集与真子集



子集：

A ⊆ B

A 中的每个元素都属于 B



真子集：

A ⊂ B

A ⊆ B 且 A ≠ B



**重点！！！！！！！！！**：

**空集是任何集合的子集！**

**任意集合都是自身的子集!**



4.2 集合的运算



集合的基本运算包括交、并、补和差。



交换律：

A ∪ B = B ∪ A

A ∩ B = B ∩ A



结合律：

(A ∪ B) ∪ C = A ∪ (B ∪ C)

(A ∩ B) ∩ C = A ∩ (B ∩ C)



分配律：

A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)

A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)



德摩根律：

(A ∪ B) = A ∩ B

(A ∩ B) = A ∪ B



同一律与零律：

A ∪ ∅ = A

A ∩ ∅ = ∅



A ∪ U = U

A ∩ U = A



4.3 有序对与笛卡尔积



4.3.1 有序对



有序对：由两个元素按顺序组成的对象。

记为：(a,b)

(a,b)=(c,d) 当且仅当:a=c或b=d。



4.3.2 笛卡尔积



A × B = { (a, b) | a ∈ A 且 b ∈ B }

有序对<x,y>中的两个元素可以分别属于不同的集合，由此可以定义一种称作笛卡尔积的有序对集合。笛卡尔积也称为直积。



若A或B为空集，则：

A × B = ∅

若 |A| = m，|B| = n，则：

|A × B| = m × n
