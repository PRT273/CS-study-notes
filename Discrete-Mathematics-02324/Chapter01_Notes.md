第一章 命题与命题公式



1.1  命题

定义

命题：能够判断“真/假”的陈述句。

真值：T（真）或 F（假）



不是命题的典型例子：

疑问句：你吃饭了吗？

祈使句：请关门。

含未知变量且未限定范围：x > 0（除非说明 x 的取值域/量词）

语义模糊：他很高。（“高”缺乏明确标准，考试通常按“不可判真伪”处理）



命题变元：

用 p, q, r ... 表示命题

命题常来自自然语言的抽象，例如：

p：今天下雨

q：我带伞



1.1.2 复合命题与联结词

五个基本联结词（最常考）

¬p 否定  非 p  p （为假时真）

p ∧ q 合取 p 且 q（两者都真）

p ∨ q 析取（或）  p 或 q 至少一个真（包含或） 

p → q 蕴含 若 p 则 q |（仅在 p 真 q 假时为假）

p ↔ q 等价  p 当且仅当 q  （两者真值相同 ）



注意：离散数学里 `∨` 默认是“包含或”，不是“异或”。



1.2 命题公式的等值演算

定义:

等值：两个命题公式在任意真值赋值下的真值都相同。

（也可说：命题公式 A 与 B 等值，或 A 与 B 等值。）



说明：

等值不是“有时相同”，而是永远相同。



判断等值的两种方法：



真值表法

适合：变元少（一般 1–3 个）

做法：把两公式真值表列出，对每一行真值逐项比较，完全一致则等值。



等值演算法

利用常用等值律，对公式逐步变形，直到得到同一个形式或明显等值的形式。



常用等值律



（1）交换律

p ∧ q 与 q ∧ p 等值

p ∨ q 与 q ∨ p 等值



（2）结合律

(p ∧ q) ∧ r 与 p ∧ (q ∧ r) 等值

(p ∨ q) ∨ r 与 p ∨ (q ∨ r) 等值



（3）分配律（**重点**）

p ∧ (q ∨ r) 与 (p ∧ q) ∨ (p ∧ r) 等值

p ∨ (q ∧ r) 与 (p ∨ q) ∧ (p ∨ r) 等值



（4）德摩根律**（必考）**

¬(p ∧ q) 与 ¬p ∨ ¬q 等值

¬(p ∨ q) 与 ¬p ∧ ¬q 等值



易错点：否定要分配到每个命题变元上，同时 ∧/∨ 互换。



（5）双重否定律

¬¬p 与 p 等值



（6）同一律 / 零律

p ∧ T 与 p 等值

p ∨ F 与 p 等值

p ∨ T 与 T 等值

p ∧ F 与 F 等值



（7）幂等律

p ∧ p 与 p 等值

p ∨ p 与 p 等值



（8）吸收律

p ∨ (p ∧ q) 与 p 等值

p ∧ (p ∨ q) 与 p 等值



蕴含与等价的常用等值变形



消去蕴含:

p → q 与 ¬p ∨ q 等值

做等值演算时，看到 → 通常先消去。



逆否命题等值

p → q 与 ¬q → ¬p 等值



等价命题的分解

p ↔ q 与 (p → q) ∧ (q → p) 等值

并且还与下式等值：

p ↔ q 与 (p ∧ q) ∨ (¬p ∧ ¬q) 等值



1.3 联结词完备集

定义：

联结词集合 S 称为完备集：

用 S 中的联结词可以表示任意命题公式（等价地：可以表示任意真值函数）。



说明：

完备的意思是：

只靠这一套联结词，就能“拼”出所有逻辑运算。



常见结论



 {¬, ∧, ∨} 是完备集

原因：常用公式都能写出来（蕴含、等价都可由它们表示）



 {¬, ∧} 是完备集

因为：

p ∨ q 与 ¬(¬p ∧ ¬q) 等值（由德摩根律得到）

所以有 ¬ 和 ∧ 就能造出 ∨，从而完备。



 {¬, ∨} 是完备集

同理：

p ∧ q 与 ¬(¬p ∨ ¬q) 等值

所以有 ¬ 和 ∨ 就能造出 ∧，从而完备。



 {→, ¬} 是完备集（常考）

因为：

p ∨ q 与 ¬p → q 等值

且有 ¬，还能构造其它运算，所以完备。



单独 {∧}、单独 {∨}、单独 {→} 都不完备

直觉：没有否定就表达不了“反过来”的真值情况（真值函数表达受限）。
