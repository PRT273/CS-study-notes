第三章 谓词逻辑



3.1 谓词的概念与表示



主语x，这是一个变量，称为**个体词**。

谓词：表示“对象具有某种性质或关系”的表达式。



举例：

P(x)：x 是学生

Q(x)：x 通过考试

R(x, y)：x 喜欢 y



个体与论域

个体：讨论的对象

人、数、集合里的元素等

论域：个体取值的范围



例如：

x ∈ 全体学生

x ∈ 自然数

x ∈ 某个班级



两种基本量词：全称量词和存在量词。



3.2 量词与合式公式



个体常量：a, b, c

个体变量：x, y, z

谓词符号：P(x), Q(x,y)

量词：

全称量词：∀x

存在量词：∃x



合式公式的核心判断：结构合法



递归定义：

若 P(x1,...,xn) 是谓词 → 是 WFF

若 A, B 是 WFF →



¬A

A ∧ B

A ∨ B

A → B



若 A 是 WFF →

∀x A

∃x A



合式 ≠ 有意义

合式 = 语法对



自由变量与约束变量：

被量词管住的变量 → 约束变量

没被量词管住的变量 → 自由变量



一个个体变元既可以是约束出现，也可以是自由出现！



3.3 谓词演算的等价式和蕴含式



**核心规则：**

**¬∀x P(x)  ≡  ∃x ¬P(x)**

**¬∃x P(x)  ≡  ∀x ¬P(x)**



语言解释\*\*（非常重要）\*\*

“不是所有 x 都满足 P”

↔ “存在一个 x 不满足 P”



**可交换：**

**∀x (P(x) ∧ Q(x)) ≡ (∀x P(x)) ∧ (∀x Q(x))**

**∃x (P(x) ∨ Q(x)) ≡ (∃x P(x)) ∨ (∃x Q(x))**



**不可交换：**

**∀x (P(x) ∨ Q(x))  ≠  (∀x P(x)) ∨ (∀x Q(x))**

**∃x (P(x) ∧ Q(x))  ≠  (∃x P(x)) ∧ (∃x Q(x))**



3.4 前束范式



前束范式＝所有量词都提前，作用域延伸到整个公式的末尾。



例：

Q1 x1 Q2 x2 ... Qn xn (量词无的矩阵)
