第七章 格与布尔代数



7.1 格的基本概念



一、偏序关系回顾



设集合 A 上定义关系 ≤ 。



若 ≤ 满足：

1）自反性：a ≤ a

2）反对称性：若 a ≤ b 且 b ≤ a，则 a = b

3）传递性：若 a ≤ b 且 b ≤ c，则 a ≤ c



则称 ≤ 为 A 上的偏序关系，

(A, ≤) 称为偏序集。



二、上界与下界



设 (A, ≤) 为偏序集，B ⊆ A。



若 x ∈ A 且对任意 b ∈ B 有：

b ≤ x



则称 x 为 B 的上界。



若 y ∈ A 且对任意 b ∈ B 有：

y ≤ b



则称 y 为 B 的下界。



三、最小上界与最大下界



设 B ⊆ A。



若 x 是 B 的上界，且任意上界 y 都满足 x ≤ y，

则称 x 为 B 的最小上界，记为：



sup B 或 b1 ∨ b2



若 y 是 B 的下界，且任意下界 z 都满足 z ≥ y，

则称 y 为 B 的最大下界，记为：



inf B 或 b1 ∧ b2



四、格（Lattice）的定义



设 (A, ≤) 为偏序集。



若对任意 a, b ∈ A：

a 与 b 的最小上界和最大下界都存在，



则称 (A, ≤) 为一个格。



五、格的运算表示



在格中定义两种运算：



a ∨ b：a 与 b 的最小上界（并）

a ∧ b：a 与 b 的最大下界（交）



于是格也可表示为代数系统：

(A, ∨, ∧)



格的基本性质:



在任意格 (A, ∨, ∧) 中，以下性质成立：



（1）交换律  

a ∨ b = b ∨ a  

a ∧ b = b ∧ a  



（2）结合律  

(a ∨ b) ∨ c = a ∨ (b ∨ c)  

(a ∧ b) ∧ c = a ∧ (b ∧ c)



（3）吸收律  

a ∨ (a ∧ b) = a  

a ∧ (a ∨ b) = a



（4）幂等律  

a ∨ a = a  

a ∧ a = a



7.2 有界格与分配格



一、有界格



若格 (A, ∨, ∧) 中存在元素 0 和 1，使得：



对任意 a ∈ A：

0 ≤ a ≤ 1



则称该格为有界格。



其中：

0 称为最小元  

1 称为最大元



二、分配格



若格中运算满足分配律：



a ∧ (b ∨ c) = (a ∧ b) ∨ (a ∧ c)

a ∨ (b ∧ c) = (a ∨ b) ∧ (a ∨ c)



则称该格为分配格。



7.4 布尔代数（Boolean Algebra）



一、布尔代数的定义



设代数系统 (A, ∨, ∧, ¬, 0, 1)。



若满足：

1）(A, ∨, ∧) 是分配格

2）存在最小元 0 和最大元 1

3）对任意 a ∈ A，存在补元 ¬a，使：

&nbsp;  a ∨ ¬a = 1

&nbsp;  a ∧ ¬a = 0



则称该系统为布尔代数。



二、布尔代数的基本规律



（1）交换律  

a ∨ b = b ∨ a  

a ∧ b = b ∧ a  



（2）结合律  

(a ∨ b) ∨ c = a ∨ (b ∨ c)  

(a ∧ b) ∧ c = a ∧ (b ∧ c)



（3）分配律  

a ∧ (b ∨ c) = (a ∧ b) ∨ (a ∧ c)  

a ∨ (b ∧ c) = (a ∨ b) ∧ (a ∨ c)



（4）德摩根律  

¬(a ∨ b) = ¬a ∧ ¬b  

¬(a ∧ b) = ¬a ∨ ¬b



三、常见布尔代数实例



（1）逻辑代数  

{T, F}  

∨ 对应“或”，∧ 对应“且”，¬ 对应“非”



（2）集合代数  

幂集 P(A)  

∪、∩、补集



（3）二值布尔代数  

{0,1}  

与、或、非运算
