第六章  代数系统的一般概念



6.1 代数系统



一、基本概念



代数系统：

由一个非空集合和定义在该集合上的运算组成的系统。



一般记为：

(A, ∘) 或 (A, ∘, \*)



其中：

A：非空集合，集合中的元素称为“元”

∘、\*：定义在 A 上的运算





二、二元运算



在集合 A 上的二元运算是一个映射：

∘ : A × A → A



含义：

任取 a, b ∈ A，

运算结果 a ∘ b 仍属于 A。





三、封闭性



若对任意 a, b ∈ A，都有：

a ∘ b ∈ A



则称运算 ∘ 在 A 上是封闭的。



封闭性是代数系统成立的基本前提。





四、常见运算性质



（1）交换律  

a ∘ b = b ∘ a



（2）结合律  

(a ∘ b) ∘ c = a ∘ (b ∘ c)



（3）单位元（幺元）  

若存在元素 e ∈ A，使得对任意 a ∈ A：

a ∘ e = e ∘ a = a



则称 e 为运算 ∘ 的单位元。  

单位元若存在，则唯一。



（4）逆元  

设 e 是单位元。  

若对某个 a ∈ A，存在 b ∈ A，使得：

a ∘ b = b ∘ a = e



则称 b 为 a 的逆元。



（5）零元（吸收元）  

若存在元素 0 ∈ A，使得对任意 a ∈ A：

a ∘ 0 = 0 ∘ a = 0



则称该元素为零元。





注意：

单位元与零元不是同一个概念。

单位元表示“不改变”，零元表示“吸收一切”。

是否存在单位元、逆元取决于具体运算。





6.2 半群与群



一、半群（Semigroup）



定义：

若代数系统 (A, ∘) 满足：

1）运算 ∘ 在 A 上封闭  

2）运算 ∘ 满足结合律  



则称 (A, ∘) 为一个半群。



半群的特点：

不要求单位元  

不要求逆元  

只关心“能不能一直算”和“算的顺序是否无关紧要”



例子：

(Z, +) 是半群  

(N, +) 是半群  

(N, ×) 是半群





二、幺半群（Monoid）



若半群 (A, ∘) 中存在单位元，

则称其为幺半群。



例子：

(Z, +)，单位元为 0  

(N, ×)，单位元为 1





三、群（Group）



定义：

若代数系统 (A, ∘) 满足：

1）运算封闭  

2）运算满足结合律  

3）存在单位元  

4）每个元素都存在逆元  



则称 (A, ∘) 为一个群。



群的结构特点：

可以来回运算  

每个操作都可以被撤销  

代数结构高度稳定



例子：

(Z, +) 是群  

单位元：0  

逆元：-a  



(Q\*, ×) 是群  

单位元：1  

逆元：1/a





四、交换群（Abelian Group）



若群 (A, ∘) 还满足：

a ∘ b = b ∘ a



则称该群为交换群。



群的基本性质：



本节不再引入新结构，主要讨论群中元素和运算的一些基本结论。



一、单位元与逆元的性质



（1）单位元唯一性  

在群 (A, ∘) 中，单位元若存在，则唯一。



说明：  

若 e 和 e' 都是单位元，则  

e = e ∘ e' = e'



（2）逆元唯一性  

在群中，每个元素的逆元若存在，则唯一。



说明：  

若 b 和 c 都是 a 的逆元，则  

b = b ∘ e = b ∘ (a ∘ c) = (b ∘ a) ∘ c = e ∘ c = c



因此逆元唯一。



（3）单位元的逆元是自身  

e 的逆元是 e 本身。



（4）逆元的逆元  

若 b 是 a 的逆元，则 a 是 b 的逆元。





二、消去律（群的重要性质）



在群 (A, ∘) 中，消去律成立：



左消去律：  

若 a ∘ b = a ∘ c，则 b = c



右消去律：  

若 b ∘ a = c ∘ a，则 b = c



说明：  

消去律依赖于“逆元”的存在，因此只在群中成立。





三、方程在群中的解



在群 (A, ∘) 中，对任意 a, b ∈ A：



方程  

a ∘ x = b  

有唯一解：



x = a^{-1} ∘ b



同理，方程  

x ∘ a = b  

也有唯一解。



说明：  

这体现了群中“运算可逆”的本质。





子群（Subgroup）：



一、子群的定义



设 (A, ∘) 是一个群，H 是 A 的非空子集。



若 H 在运算 ∘ 下本身构成一个群，  

则称 H 是 A 的一个子群，记作：



H ≤ A





二、子群判定定理（非常重要，考试必考）



设 (A, ∘) 是群，H ⊆ A 且 H ≠ ∅。



若 H 满足以下条件，则 H 是 A 的子群：



1）对任意 a, b ∈ H，有  

a ∘ b ∈ H   （封闭）



2）对任意 a ∈ H，有  

a^{-1} ∈ H  （对逆元封闭）



说明：  

在群中，不需要单独验证单位元和结合律。





三、子群判定的常用等价形式



常用判定法（记住这个就够）：



H 是子群  

当且仅当  

对任意 a, b ∈ H，都有：



a ∘ b^{-1} ∈ H





四、常见子群举例



（1）整数群的子群  

在 (Z, +) 中：



nZ = { nk | k ∈ Z }  

是 Z 的子群。



例如：  

2Z, 3Z, 5Z 等都是 Z 的子群。



（2）平凡子群  

任意群都有两个显然的子群：



{e}  

和  

A 本身



其中 {e} 称为平凡子群。
