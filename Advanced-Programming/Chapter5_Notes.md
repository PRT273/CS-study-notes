第五章 数组



5.1：一维数组



一维数组的定义：

一般形式如下：数据类型符 数组名1\[长度1],数组名2\[长度2],...;

“数据类型符”是指数组元素的数据类型。“长度”必须是整型的常量表达式，其中不能出现变量。

下标是从零开始顺序编号。例：

int a\[10];

float score\[50];

char name\[20];



5.1.2 一维数组的引用

引用方法：数组名\[下标]。

保证数组下标不越界十分重要!

一维数组的内存字节数=数组长度x每一元素占用的字节数。



5.1.3 一维数组的初始化

格式如下：数据类型符 数组名\[长度]=\[初值表],...;。

完全初始化：int a\[5] = {1, 2, 3, 4, 5};。

部分初始化（剩余元素自动为0）：int b\[5] = {1, 2}; 等价于：{1, 2, 0, 0, 0}。

自动推断数组长度：int c\[] = {3, 6, 9}; 自动判断长度为3。
