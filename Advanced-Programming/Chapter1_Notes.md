第 1 章：计算机概述（笔记）



今天主要看了三块内容：计算机发展、计算机语言、算法。

我把重点记一下，算是自己的理解，不一定完全准确，但以后用得到。



1.1 计算机是怎么发展起来的



最早的计算机是 1946 年的 ENIAC（电子管），大、烫，也耗电。

后来冯诺依曼提出了“存储程序”的思想，把程序和数据都放到内存里，这样电脑可以按照指令自己跑。

计算机结构大概就是：  控制器 + 运算器 + 存储器 + 输入设备 + 输出设备

  现在的电脑其实还是这个结构，只不过换成了更小、更强的硬件。



大概发展过程：

\- 机械时代 → 电子管 → 晶体管 → 集成电路 → 超大规模集成电路

每一代都越来越快、越来越小。



1.2 计算机语言



其实就是不同“层级”的语言：

机器语言（0 和 1）

人根本没法直接用，但电脑最喜欢。



汇编语言

用符号替代 0 和 1，例如 MOV、ADD 这种。

比机器语言好一点，但还是挺难读。



高级语言

Java、Python、JavaScript、C语言

人能看懂，电脑要通过“编译器/解释器”翻译成机器语言。



顺便记一下：

C/Java 这种是编译型 先编译再运行

Python 是解释型 一边运行一边解释



1.3 算法（感觉挺重要的）



老师的定义比较正式，我自己的理解是：



解决一个问题的步骤，必须清楚、有限，而且能真正跑得通



算法有几个特点：

有穷性 → 不能无限循环

确定性 → 每一步都写清楚

输入输出 → 输入可以没有，但输出至少有一个

可行性 → 每一步都能实际做出来（不是幻想）



算法的几种写法：

自然语言（文字描述）

伪代码（这种在教材里最常见）

流程图（图形 箭头那种）
